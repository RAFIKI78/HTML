Dinosaurios mas grandes

Argentinosaurus
Longitud: 30–40 metros / Altura (hasta los hombros): 7–8 metros / Peso: 60–100 toneladas / Época: Cretácico Superior (hace unos 94 millones de años) / Lugar: Argentina
Patagotitan mayorum
Longitud: ~37 metros / Altura (hasta los hombros): ~6 metros / Peso: ~60–70 toneladas / Época: Cretácico Superior / Lugar: Argentina
Dreadnoughtus schrani
Longitud: ~26 metros (posiblemente más) / Altura (hasta los hombros): ~6 metros / Peso: ~50–65 toneladas / Época: Cretácico Superior / Lugar: Argentina
Supersaurus vivianae
Longitud: ~33–39 metros / Altura: ~5–6 metros / Peso: ~35–40 toneladas / Época: Jurásico Superior / Lugar: EE.UU. (Colorado)
Sauroposeidon proteles
Longitud: ~30–34 metros / Altura (cuello levantado): hasta 17 metros / Peso: ~50 toneladas / Época: Cretácico Inferior / Lugar: EE.UU. (Oklahoma, Texas)
